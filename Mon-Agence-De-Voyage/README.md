# Mon agence de voyage

## Description
Mini-projet réalisé pour découvrir les balises HTML de base.  
Il contient un exemple simple d’intégration de vidéo et d’audio pour illustrer le contenu multimédia sur une page web.

## Objectif
Pourquoi j’ai réalisé ce projet :  
- Apprendre le fonctionnement des balises HTML `<audio>` et `<video>`  
- Tester l’insertion de contenus externes dans une page web  

## Fonctionnalités
- Lecture d’une vidéo intégrée avec contrôle utilisateur  
- Lecture d’un fichier audio avec contrôle et boucle  

## Technologies utilisées
- HTML  

## Ce que j’ai appris
- Compréhension de la structure globale d’une page HTML  
- Utilisation des balises multimédia : `<video>` et `<audio>`  
- Difficultés rencontrées avec les balises `<img>` et ses attributs, `<a>` pour les liens, `<figcaption>`  
- Apprentissage des listes HTML : `<ul>`, `<li>`, `<ol>`  

## Améliorations possibles
- Projet purement exercice, aucune amélioration prévue
