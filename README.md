# Projets FreeCodeCamp – Apprentissage guidé

Ce dépôt contient les projets réalisés dans le cadre des parcours **FreeCodeCamp**.

## Objectif du dépôt
- Suivre les exercices et projets demandés par FreeCodeCamp
- Comprendre les bases du HTML / CSS / JavaScript
- Valider les concepts fondamentaux sans personnalisation excessive

## Contenu
Chaque dossier correspond à un projet ou exercice proposé par FreeCodeCamp, réalisé selon les consignes de la plateforme.

## Remarque importante
Ces projets ne sont **pas destinés à être utilisés comme portfolio**.
Ils servent uniquement de support d’apprentissage structuré.

## Technologies utilisées
- HTML
- CSS
- JavaScript (selon les projets)

## Auteur
Projet réalisé dans un cadre d’apprentissage personnel.
