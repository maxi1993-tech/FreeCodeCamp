Mon second projet html : Une page de librairie (freecodecamp)

#  XYZ Bookstore (En cours d'apprentissage)

Ce projet est mon premier exercice complexe sur la structure HTML. Je l'ai réalisé via **freeCodeCamp** pour me confronter à des mises en page professionnelles.

##  Ce que j'ai pratiqué :
- **Structure de base** : Création de conteneurs (`div`) pour organiser les livres.
- **Boutons interactifs** : Ajout de boutons avec des classes pour le futur design.
- **Lecture de consignes** : Apprendre à suivre un cahier des charges technique précis.

##  Ce que je suis en train de consolider :
Je travaille encore sur la compréhension totale des **IDs uniques** par rapport aux **classes CSS**, ainsi que sur l'imbrication profonde des éléments (balises dans des balises). Mon objectif est de rendre ce code de plus en plus propre.

##  Prochaine étape :
Appliquer du CSS pour transformer ces blocs bleus et blancs en une véritable boutique visuelle !
