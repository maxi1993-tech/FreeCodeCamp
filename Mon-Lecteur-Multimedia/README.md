# Mon Lecteur Multimédia

## Description
Projet réalisé pour explorer l’intégration de contenu multimédia dans une page HTML.  
Il contient un exemple de vidéo et d’audio avec contrôle utilisateur et lecture en boucle.

## Objectif
Pourquoi j’ai réalisé ce projet :  
- Apprendre à utiliser les balises HTML `<video>` et `<audio>`  
- Tester l’insertion de fichiers multimédias externes sur une page web  

## Fonctionnalités
- Lecture d’une vidéo intégrée avec contrôle (`play/pause`)  
- Lecture d’un fichier audio avec contrôle et boucle (`loop`)  

## Technologies utilisées
- HTML  

## Ce que j’ai appris
- Compréhension des balises multimédia `<video>` et `<audio>`  
- Utilisation des attributs `controls`, `loop` et `width`  
- Structure des sections HTML (`<section>`, `<h1>`, `<h2>`)  
- Gestion de sources externes pour les médias  

## Améliorations possibles
- Ajouter plusieurs vidéos ou pistes audio pour créer une petite playlist  
- Rendre le projet responsive pour les petits écrans  
- Ajouter des descriptions ou légendes pour chaque média
