#  Ma Recette de Lasagnes Maison

Bienvenue sur mon premier projet de développement web ! Ce site présente ma recette préférée de lasagnes, structurée proprement en HTML.

##  Objectifs du projet :
Ce projet m'a permis de pratiquer les bases fondamentales du Web :
* **Listes ordonnées (`<ol>`)** : Pour les étapes de la préparation.
* **Listes à puces (`<ul>`)** : Pour la liste des ingrédients.
* **Titres (`<h1>`, `<h2>`)** : Pour hiérarchiser les informations.
* **Images** : Pour rendre la recette appétissante !

##  Ingrédients clés :
* Feuilles de lasagnes fraîches
* Sauce Bolognaise maison
* Béchamel onctueuse
* Beaucoup de fromage !

##  Ce que j'ai appris :
En codant cette page, j'ai compris comment transformer un document texte classique en une **page web structurée** que n'importe quel navigateur peut lire.
