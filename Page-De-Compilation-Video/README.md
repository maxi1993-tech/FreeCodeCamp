# Video Compilation Page – Data Analyst

## Description
Ce projet est une page HTML présentant une compilation de vidéos pédagogiques sur le métier de Data Analyst.  
La page structure le contenu en sections claires avec intégration de vidéos YouTube via iframe.

## Objectif
Pourquoi j’ai réalisé ce projet :
- Comprendre et appliquer la structure sémantique HTML (`main`, `section`, hiérarchie des titres).
- Apprendre à intégrer des vidéos avec la balise `<iframe>`.
- Respecter un cahier des charges précis (ordre des éléments, attributs obligatoires).

## Fonctionnalités
- Structure HTML organisée avec un seul élément `main`.
- Trois sections thématiques (SQL, Python, Power BI).
- Intégration de vidéos YouTube via `iframe` avec attributs obligatoires (`src`, `title`, `width`, `height`).

## Technologies utilisées
- HTML5

## Ce que j’ai appris
- L’importance de la hiérarchie des titres (`h1`, `h2`).
- Comment utiliser correctement une URL YouTube en format `/embed/`.
- L’utilité de l’attribut `title` pour l’accessibilité.
- Respect strict d’un cahier des charges technique.

## Améliorations possibles
- Ajouter du CSS pour améliorer la mise en page.
- Rendre les vidéos responsive.
- Améliorer la rédaction du contenu descriptif.
- Ajouter une navigation interne.
